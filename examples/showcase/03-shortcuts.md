# Shortcuts

Everything peek does, reachable from the keyboard. No menu hunting.

## Files

| Key | Action |
|---|---|
| `⌘O` | Open file |
| `⌥⌘O` | Open folder |
| `⌘R` | Reload |
| `⌘W` | Close window |

## Navigation

| Key | Action |
|---|---|
| `⌘F` | Find in page |
| `⌘⇧O` | Toggle outline (TOC) |
| `⌘↑` / `⌘↓` | Top / bottom |
| `Space` | Page down |

## Sidebar (vim-style)

| Key | Action |
|---|---|
| `j` / `k` | Move down / up |
| `h` / `l` | Collapse / expand |
| `gg` / `G` | Top / bottom |
| `Enter` | Open |

## View

| Key | Action |
|---|---|
| `⌘=` / `⌘-` / `⌘0` | Zoom in / out / reset |
| `⌘⇧D` | Toggle light / dark |
| `⌘P` | Print |
| `⌘⇧E` | Export PDF |
| `⌘⌃F` | Fullscreen |
